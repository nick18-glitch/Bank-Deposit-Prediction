# Bank Deposit Prediction

This project uses a **Machine Learning classification model** to predict whether a client will subscribe to a bank term deposit based on customer and campaign information.

## Dataset

The dataset contains client details such as age, job, marital status, education, balance, contact type, and previous marketing campaign outcomes.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## Steps Performed

* Data cleaning and preprocessing
* Handling missing values
* Encoding categorical variables
* Train-test split
* Model training and prediction

## Model Evaluation

The model performance was evaluated using:

* Confusion Matrix
* Classification Report

## Goal

To understand how machine learning can be used to **predict customer responses in marketing campaigns**.
